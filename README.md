# QuestAutoComplete

QuestAutoComplete is a Discord plugin that auto complete your discord quests with ease without actually doing them!
## Features
- Automatically accept quests and auto-complete them
- Tricks discord client into thinking you're playing the game
- Auto-completes & skips video quests (only for video ad quests)
- Auto-Complete multiple quests at a time
## Settings
- **ShowNotifications** : Notify you with current status of the quest, progress precentage & quest name and more
- **NotifcationDuration** : how long the notifcations stay on screen (seconds)
- **AutoResumeAfterReload** : Resume to the prevouise quest and contiune completing it after closing discord & reloading it
- **AutoAcceptAndComplete** : Auto accept new discoverd quests & complete them Automatically **BETA**
- **AllowMultipleQuests** : Auto complete more than a one quest at a time (use with caution, may get you banned) **BETA**
- **AutoAcceptDelay** : delay time to go accept a new quest & complete it (to prevent rate limit & suspicious activity)

## Installation

1. download the Plugin zip file then move it to the vencord userplugins folder **vencord/src/userpllugins**
2. Extract all the plugins files in the same userplugins folder
3. Open vencord folder inside a terminal
4. Run the commands **pnpm build** and then **pnpm inject**
3. Enable **QuestAutoComplete** from the Vencord plugins list.

## Notes

- Enbaling **AllowMultipleQuests** can get your account **Flagged as a bot**, use with caution and the knoweldge of knowing that you can get terminated, to avoid this use the **AutoAcceptDelay**
